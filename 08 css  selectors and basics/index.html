Selectors in css

selectors are used to find the element to whose property should be styled

Types of css selectors

* is universal Selectors

element selector such as p,div,nav,img,span,h1,section etc

. is class selector and a element can have multiple classes separated by space

# is id selector and it is unique and can have only one id to one element

css group selectors means you can write same style to multiple elements separated by space at a time
